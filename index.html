PRIDE

HTML

A Javascript library to create prideful text!

HOW TO USE IT

            
                // In the head tag
                <script src="pride-html.js"></script>
                // ...

                // At the end of the body tag
                const ph = new PrideHTML();

                ph.setAngle(90); // optional
                ph.setSpeed(1.5); // optional

                // Parameters are the
                // 1. CSS selector for the element.
                // 2. The flag to be prideful with.
                ph.colorTextElement(".your-class", "traditional");

WHAT FLAGS ARE SUPPORTED?

The supported flag keywords are:

  • traditional

  • philadelphia

  • progress

  • trans

  • non-binary

  • intersex
